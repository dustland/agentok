# Studio Features

Agentok Studio is a visual builder for [AG2](https://github.com/ag2ai/ag2) multi-agent applications. Design workflows on a canvas, run chats in the browser, and export Python you can run anywhere.

## What you can do

| Area | What it does |
| --- | --- |
| [Workflow Canvas](/docs/guides/build/workflow-canvas) | Drag agents onto a canvas and configure their behavior |
| [Conversation Patterns](/docs/guides/build/conversation-patterns) | Connect agents and define how they talk to each other |
| [Tools](/docs/guides/build/tools) | Create reusable tools and attach them to conversations |
| [Code & Debugging](/docs/guides/build/code-and-logs) | Inspect generated Python, flow JSON, and runtime logs |
| [Templates & Sharing](/docs/guides/build/templates) | Publish workflows and let others fork or chat with them |

## Studio areas

After signing in at [studio.agentok.ai](https://studio.agentok.ai):

- **Projects** — create and edit workflows on the canvas
- **Chats** — run and review live multi-agent sessions
- **Tools** — manage custom tools and variables
- **Discover** — browse and fork published templates
- **Settings** — configure models, API keys, and account

## Quick path

New to the studio? Start with [Getting Started](/docs/getting-started) to build a Hello World workflow, then read the feature guides above for deeper coverage.

For programmatic access, see [API](/docs/guides/using/api) and [Embed](/docs/guides/using/embed).
